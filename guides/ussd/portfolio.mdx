### Accessing the Portfolio View

<Tip>
  This request is made from the `Home` menu view.
</Tip>

```bash
  curl --request POST \
    --url https://api.staging.pipevest.com/v1/ussd/process
    ...
    --data 'sessionId=<session-id>'
    --data 'choice=2'
```
